IIFE Example

Open up the browser console, you'll see 'I am an IIFE' in the logs.

Because, it's an IIFE(Immediately Invoked Function Expression):

        (function () {
            console.log("I am an IIFE");
        })();

To be able to see 'I am NOT an IIFE' text, you need to invoke notIIFE() function because it's NOT an IIFE.

To invoke, write notIIFE() to the console and hit enter.
